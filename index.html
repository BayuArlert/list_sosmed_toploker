⚙️ Panduan Setup Google Credentials
⚠️ Harus dilakukan sekali sebelum pertama kali sync ke Google Sheets.
  1. Buka console.cloud.google.com → buat project baru
  2. Aktifkan Google Sheets API (Library → cari "Sheets API" → Enable)
  3. Buka IAM & Admin → Service Accounts → Create Service Account
  4. Setelah dibuat, klik akun tersebut → tab Keys → Add Key → JSON → Download
  5. Rename file yang didownload menjadi credentials.json dan letakkan di folder project ini
  6. Copy email service account (format: nama@project.iam.gserviceaccount.com)
  7. Buka Google Spreadsheet → klik Share → paste email tadi → beri akses Editor
  8. Restart server → status Credentials akan berubah menjadi ✅
